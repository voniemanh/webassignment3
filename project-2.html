Goals

Goal 1

The primary goals of this project are to develop a web application that allows users to register their pets' basic information, including name, ID, weight, type of pet, vaccination status, BMI, registration date, and filter healthy pets.

Goal 2

The aim is to create an efficient and straightforward platform for managing and accessing pet records.

Scopes

IN

This project involves collaboration with pet owners and veterinarians to ensure that the information collected is relevant and useful.

OUT

It also entails gathering feedback from users to improve the application's functionality and user experience.

Primary

Implementing a user-friendly registration form for pet information.
Designing a clean and intuitive layout.

Adding features such as filtering healthy pets based on vaccination status and BMI
Ensuring the application is accessible and responsive across different devices.
